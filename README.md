# Magisk Malayalam Fonts

Magiks Module to change default Malayalam font.

## Supported Fonts and Combinations
- Manjari
- Gayathri
- Gayathri for UI + Manjari for Content
- Rachana
- Meera
- Anjali Old Lipi
- Suruma
- Raghu Malayalam
- Dyuthi
- Keraleeyam
- Uroob
- Chilanka
- Karumbi
